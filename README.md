# microbiome-abtract
scholar.r should download some number of abstracts from PLOS.
One sample set of text is contained in microbiome_plos_500.txt (approximately 80,000 words from 500 abstracts).
The markov_two_abstract.pl is a 2nd order markov model that makes an abstract from which you cna then test hypotheses. It takes two arguments: the first is an organism (or thing) that it will attempt to make your abstract discuss; the second is the number of words in the abstract.
